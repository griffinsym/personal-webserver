High Level Description

This system was my solo project during my internship at SOLV Energy and effectively allows the elimination of survey feathers (physical markers placed by a survey team) by creating a digital navigation tool to guide distribution equipment operators to a location. The intended accuracy of this system was to be within 2ft of the true location but we had significantly lower average drop deviations. The system involved a base station, a modular rover kit, and a digital navigation aid. We also prototyped a system to record when and where something is distributed.

Core Functions and Operations

This system has 3 primary operations. They are as follows:
The base station. This system is independently powered (in our case with a solar panel) and sits at a known location. This location can be quickly updated using a simple update script that flashes it directly to the base station's non-volitile memory. The base station has a single antenna that repeatedly gets it's location and compares it against it's known coordinates. It then takes the difference and transmits it via a radio module to the rover.
The rover. The rover is the portion of this system that is mounted to the vehicle and moves around. The rover has 2 antennas that repeatedly get their respected locations. The rover also takes in the broadcast correction information provided by the base station. By subtracting the correction value from primary antenna's location we get a more accurate coordinate value. Also, by subtracting these locations we get a directional vector telling us which way the vehicle is facing with a few degrees of accuracy. Additionally, using the height information of the two antennas allows us to differentiate for the pitch of the vehicle (in the front to back direction only). The rover then outputs the corrected coordinates, heading and pitch.
The navigation aid. This system is basically just a computer that takes an input from the rover, processes the information and visually displays it to the user using a custom tailored mapping software. The operator is also given a number of capabilities including selecting a to-point navation line, multi-point active selection, automatic distribution recording and reactive optimization options.
